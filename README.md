# pima-indians-diabetes
Machine Learning / Deep Learning Model to outcome Population of Pima Indians with Diabetes

This project uses the Pima Indians Diabetes dataset (UCI ML Repository) to build supervised learning models that predict the likelihood of diabetes onset based on medical attributes. The goal is to demonstrate how machine learning/deep learning and explainable AI can support healthcare decision-making by identifying risk factors and improving early intervention strategies.
